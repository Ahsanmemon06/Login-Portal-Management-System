# Login Portal Management System

A simple Login Portal Management System built using Python and Streamlit. This application allows users to:

- Add new login credentials
- View saved logins
- Search login credentials
- Update passwords
- Delete login records

## Technologies Used
- Python
- Streamlit

## Storage
- Local text file (`logindatafile.txt`)

## Features
- Simple and user-friendly interface
- CRUD operations (Create, Read, Update, Delete)
- Lightweight and beginner-friendly project

Note: This project is for learning purposes only. Passwords are stored in plain text and are not encrypted
